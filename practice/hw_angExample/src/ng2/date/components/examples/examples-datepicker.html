• [small]: boolean - render control in small or default size (default is false)
  • [placement]: 'top-left' | 'top-right' | 'bottom-left' | 'bottom-right' - where to show calendar dropdown (default is 'bottom-left')
  • [format]: string - format how to display selected date in input (default is LL)
  • [modelFormat]: string - if defined - ngModel value will be string in provided format, if not set - value will be moment.Moment object
  • [placeholder]: string - text for empty input
  • [startOnMonday]: boolean - if to render calendar starting week from Monday (true) or Sunday (false) (default is true)
  • [editTime]: boolean - if to display time edit controls (default is false)
  • [minDate]: moment.Moment - optional, if set - defines min date allowed for selection
  • [maxDate]: moment.Moment - optional, if set - defines max date allowed for selection
  • [activeMonth]: moment.Moment - optional, if set - activates month in calendar. Will be overwritten right after Calendar value changes

Component implements ControlValueAccessor - i.e. supports [(ngModel)]
ngModel value is either moment.Moment date object or string in format defined by modelFormat param
