cad-radio-button-group

  • [disabled]: boolean - Is radio button group disabled
  • [buttonDirection]: RadioButtonDirection - 'in-row' | 'in-column'
  • [name]: string - Unique name for radio group
  • [(ngModel)]: any - Value for radio group
  • (onChange): EventEmitter<any> - Trigger when value is changed

  • [value]: any - Radio button value (this value is set to radio button group value if radio button is active)
  • (onChange): EventEmitter<void> - trigger when radio button is selected
