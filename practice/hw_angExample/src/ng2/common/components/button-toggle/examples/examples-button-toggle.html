cad-button-toggle-group

  • [disabled]: boolean - Is toggle group disabled
  • [type]: ButtonToggleGroupType Toggle group appearance
  • [activeButton]: any - Define active button in the group
  • (onChange): EventEmitter<any> - Trigger when active button is changed

cad-button-toggle

  • [disabled]: boolean - Is button disabled
  • [value]: any - Button value (this value is set to Toggle group value if button is active)
  • (onClick): EventEmitter<void> - trigger when button is pressed
