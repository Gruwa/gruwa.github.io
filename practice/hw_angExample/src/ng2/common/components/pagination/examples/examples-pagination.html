• @Input() disabled: boolean - if component disabled (default is false)
  • @Input() autohide: boolean - if to automatically hide component when there's no or just 1 page in total (default is false)
  • @Input() currentPage: number - current page number, starts from 1
  • @Input() small: boolean - if component should be shown in smaller size
  • @Input() totalPages: number - total numer of pages
  • @Output() onChange($event: number) - event emitted on current page changes
