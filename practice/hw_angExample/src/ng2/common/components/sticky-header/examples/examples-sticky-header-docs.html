Container for content that should stick to main header when page is scrolled and show as subheaders.
Internally all inner content of the component is transferred into main header component, right below the header markup.
No need to add "display: flex" or manually listen scrollbar position to achieve "sticky" behaviour.
  • @Output() sticked = new EventEmitter<boolean> - event emitted when sticked state changes
Component is a container for content that should stick to main header when page scrolled.
All animations and visual changes between sticked/not-sticked states should be specified at inner markup and styles.
  • @Input() startHeight: number - initial (not sticked state) header height in pixels
  • @Input() endHeight: number - sticked state header height in pixels
  • @Input() stickyFrom: number - set sticked state when scroll position is higher than this value (default is 0)
  • @Output() sticked = new EventEmitter<boolean> - event emitted when sticked state changes
