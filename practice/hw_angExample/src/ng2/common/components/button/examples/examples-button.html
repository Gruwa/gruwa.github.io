• [view]: string - name of button theme
  • [wide]: boolean - if true button will stretch to container's width (default is false)
  • [small]: boolean - if true button will has smaller height size (default is false)
  • [icon]: string - name of the icon to render near text
  • [text]: string - button text, auto-translated
  • [type]: string - html button type (default is button)
  • [disabled]: boolean - if button is disabled
  • [showSpinner]: boolean - disable button and show spinner inside button
  • @Output() onClick($event: MouseEvent) - event emitted on button click.
    Why do we need it: 1. to bind click on <button>, not on <cad-button> . 2. to prevent the "click" event on disabled buttons
