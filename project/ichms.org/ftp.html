  • ФТП Сервис

    Вы можете воспользоваться нашим ФТП ресурсом. На котором Вы можете найти много интересующей Вас информации. Такой как:

    1. Электронные версии книг изданных в Лаборатории.
    2. Технические данные той или иной разработки.
    3. Фотографии из жизни и работы Лаборатории.
    4. Видео материалы снятые в Лаборатории.
    5. И многое другое.

    Для работы с FTP-сервером удобнее всего использовать специальные программы***:

    1. FTP-клиенты (например, CuteFTP, FlashFXP, FileZilla, AceFTP Free)
    2. Файл-менеджеры (например, FАR, TotalCommander, Windows Commander)
    3. Менеджеры закачки (например, FlashGet, ReGet, GetRight, Download Master)

    В этом списке Вы найдете ссылки на несколько программ, которые доступны как FREEWARE продукты

    Для соединения с сервером ftp://harbour.teremki.net.ua/ с помощью любой из этих программ в настройках соединения необходимо указать:

    1. Имя сервера - harbour.teremki.net.ua.
    2. Порт стандартный - 21.
    3. Пользователь анонимный (логин anonymous, пароль не нужен).
    4. После этого установите пассивный режим работы с сервером (passive mode).
